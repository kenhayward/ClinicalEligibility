# eligibility

The extraction pipeline that converts raw clinical-trial eligibility criteria
into structured, UMLS-coded records.

This is the single context in the repository. Its projects live under
[`src/`](src/) and [`tests/`](tests/), with the solution at
[`Eligibility.sln`](Eligibility.sln).

For the full overview - what it does, architecture, quick start, CLI commands,
configuration, deployment, and status - see the [repository README](../../README.md)
and the canonical specs under [`docs/specs/`](../../docs/specs/).
